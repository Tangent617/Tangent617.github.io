消消乐

题意

给一串数，两个相邻的和为10可以消掉，问最多消多少次。

分析

用栈保存，栈顶与当前和为10就弹出，否则压入。

代码

#include <iostream>
#include <string>
#include <stack>
using namespace std;

const int maxn = 1e6 + 5;

int main(int argc, const char * argv[])
{
    int n, a[maxn];
    string str;
    stack<int> s;
    cin >> n >> str;
    for (int i = 0; i < n; i++) {
        a[i] = str[i] - '0';
        if (!s.empty() && a[i] + s.top() == 10) {
            s.pop();
        } else {
            s.push(a[i]);
        }
    }
    cout << s.size() << endl;
    return 0;
}

case通过率：100%
